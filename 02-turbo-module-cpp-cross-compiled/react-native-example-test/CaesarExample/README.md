# CaesarExample — app de ejemplo (enfoque ②)

App RN que consume `@nicovegasr/caesar-rn` (TurboModule C++) desde el Verdaccio local, con playground de latencia async vs sync y generadores de payload.

La guía completa para arrancar emuladores, publicar al registry local y depurar está en el [README del playground](../README.md).

```bash
yarn start                          # Metro
yarn ios        # o   yarn android
```
